# iiko Счёт-фактура Бот

Telegram-бот для автоматической обработки PDF счетов-фактур: извлечение товаров, поиск в базе iiko и добавление поставки.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `.env.example` в `.env`
2. Получите токен бота у [@BotFather](https://t.me/BotFather)
3. Добавьте в `.env`:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен
   ```
4. Для поиска в iikoServer (опционально):
   ```
   IIKO_SERVER_URL=https://bufet-17-co.iiko.it
   IIKO_SERVER_LOGIN=ваш_логин
   IIKO_SERVER_PASSWORD=ваш_пароль
   ```

Без учётных данных iiko бот не сможет искать товары в номенклатуре.

## Запуск

```bash
python main.py
```

## Использование

1. Отправьте боту команду `/start`
2. Отправьте PDF файл счёта-фактуры
3. Бот извлечёт товары и покажет для проверки
4. Нажмите «Подтвердить» для добавления поставки или «Отмена»
